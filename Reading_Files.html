Reading Files

Reading Files that meet a condition

It’s pretty common for analyst to get the question, “What was ___ in 2016?” If you have 10 years’ worth of data saved in comma-delimited files, but you only want to read the 2016 data into memory, you can quickly specify which files you want to read.

# Create the list of files
file_list <- list.files(path = "../somedirectory" , pattern = ".csv", full.names = TRUE)

# subset files to only include 2016, this could also be done with regex
file_list <- file_list[substr(file_list,89,92)=="2016"]

#read the files
csvlist <- lapply(file_list, read.csv, stringsAsFactors = FALSE)

#makes cumulative dataframe, binding rows
df <- do.call(rbind, csvlist)
